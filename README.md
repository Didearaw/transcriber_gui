[![License](https://img.shields.io/badge/License-MIT-yellow)](LICENSE)

**Универсальный графический инструмент** для скачивания видео с YouTube, извлечения аудиодорожки и автоматической расшифровки речи с помощью нейросети **OpenAI Whisper**. Всё работает **полностью локально** – никаких платных API, облачных сервисов или ограничений.

---

## ✨ Возможности

- ✅ **Три режима работы** (выбираются в один клик):
  - 📥 **Скачать + расшифровать** – полный цикл.
  - ⬇️ **Только скачать** – видео (в максимальном качестве) и аудио.
  - 🎤 **Только расшифровать** – загрузите готовый аудиофайл (WAV, MP3, M4A и др.).
- ✅ **Видео в наилучшем доступном качестве** (вплоть до 4K/8K) с автоматическим объединением потоков в MP4.
- ✅ **Извлечение аудиодорожки в WAV** для качественной транскрибации.
- ✅ **Поддержка GPU (CUDA)** – автоматическое ускорение распознавания при наличии видеокарты.
- ✅ **Автообход защиты YouTube** – использование cookies из Firefox/Chrome/Edge или файла `cookies.txt`.
- ✅ **Автоматическая установка зависимостей** при первом запуске (включая `ffmpeg`, `yt-dlp`, `whisper`).
- ✅ **Удобный графический интерфейс** с прогресс-баром и информацией о статусе.
- ✅ **Чистые папки** – временные файлы автоматически удаляются, остаются только итоговые результаты.
- ✅ **Сохранение настроек** – выбранный режим и путь к cookies запоминаются.

## 📁 Структура папок

При запуске скрипт создаёт рядом с собой три папки:

```
📁 папка_со_скриптом/
├── transcriber_gui.py
├── transcriber_config.json         (настройки cookies и режима)
├── 📁 YouTube_Video/               (скачанные видео в .mp4)
├── 📁 YouTube_Audio/               (извлечённые аудиодорожки в .wav)
└── 📁 Transcriber/                 (текстовые расшифровки в .txt)
```

## 🚀 Быстрый старт

1. **Установите Python 3.8+** (если ещё не установлен).  
   При установке **обязательно отметьте** «Add Python to PATH».

2. **Скачайте скрипт** `transcriber_gui.py` и поместите его в удобную папку.

3. **Запустите скрипт** двойным кликом или из терминала:
   ```bash
   python transcriber_gui.py
   ```

4. При первом запуске скрипт **автоматически проверит и установит** все зависимости.  
   Это может занять пару минут – просто подождите.

5. Выберите режим работы:
   - Вставьте ссылку на YouTube и нажмите **«Старт»**.
   - Или выберите локальный аудиофайл для расшифровки.

6. Дождитесь окончания процесса – результат появится в соответствующих папках.

7. **ВАЖНО!** Настоятельно рекомендую установить браузер Firefox!

## 🎛 Выбор модели Whisper

В выпадающем списке доступны пять моделей:

| Модель  | Размер  | ОЗУ (примерно) | Скорость (CPU) | Точность | Рекомендация |
|---------|---------|----------------|----------------|----------|--------------|
| `tiny`  | ~75 MB  | ~400 MB        | Очень быстро   | Низкая   | Для тестов   |
| `base`  | ~145 MB | ~600 MB        | Быстро         | Средняя  | Черновики    |
| `small` | ~488 MB | ~1.2 GB        | Умеренно       | Хорошая  | **Оптимально** |

Модель скачивается **один раз** и хранится в папке `~/.cache/whisper/`.

## 🛡 Обход блокировки YouTube

Если при скачивании появляется ошибка **«Sign in to confirm you’re not a bot»**:

### 🔐 Способ 1. Использовать cookies из браузера (рекомендуется)

- Войдите в свой аккаунт YouTube в **Firefox, Chrome или Edge**.
- Скрипт автоматически извлечёт сессионные cookies из браузера.

### 🍪 Способ 2. Экспортировать cookies в файл

1. Установите расширение **«Get cookies.txt LOCALLY»**.
2. Зайдите на YouTube, нажмите на иконку расширения и выберите **«Export»**.
3. Сохраните файл `cookies.txt`.
4. В интерфейсе программы нажмите **«📁»** и укажите путь к файлу.

> **Важно**: cookies должны быть свежими. При ошибке попробуйте экспортировать файл заново.

## 🖥️ Использование GPU (CUDA)

Если у вас видеокарта NVIDIA с поддержкой CUDA и установлен PyTorch с CUDA, Whisper **автоматически** начнёт использовать GPU. Это ускорит распознавание в несколько раз.

При запуске в консоли появится сообщение:
```
Whisper будет использовать устройство: CUDA
```

Если видите `CPU` – проверьте установку PyTorch с поддержкой CUDA.

## 🔧 Устранение неполадок

### ❌ Ошибка «Sign in to confirm you're not a bot»
- Войдите в YouTube в браузере **Firefox** (наиболее стабильно работает).
- Или укажите свежий файл `cookies.txt`.

### ❌ Видео скачивается в низком качестве
- Убедитесь, что у видео есть более высокие разрешения (проверьте на YouTube).
- Если видео доступно только в 360p – скрипт скачает лучшее из доступного.

### ❌ Не устанавливается `yt-dlp` или `whisper`
- Запустите вручную:
  ```bash
  pip install -U yt-dlp openai-whisper imageio-ffmpeg
  ```

### ❌ Ошибка `FP16 is not supported on CPU`
- Это предупреждение, оно не влияет на работу. Скрипт автоматически переключается на FP32.

### ❌ После скачивания остаются лишние файлы
- Скрипт автоматически удаляет временные `.f*.mp4` и `.m4a`. Если что-то осталось – удалите вручную.
