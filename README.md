# 🎬 YouTube Transcriber & Video Downloader

Простой и мощный графический инструмент для скачивания видео с YouTube, извлечения аудиодорожки и автоматической расшифровки речи с помощью нейросети OpenAI Whisper.

**Работает полностью локально** — не зависит от платных API и облачных сервисов. Всё, что нужно — Python и этот скрипт.

---

## ✨ Возможности

- ✅ **Скачивание видео** с YouTube в наилучшем качестве (mp4).
- ✅ **Извлечение аудиодорожки** в формат `.wav` для качественной транскрибации.
- ✅ **Автоматическая расшифровка речи** на русском языке с помощью Whisper (можно выбрать модель от `tiny` до `large`).
- ✅ **Автообход защиты YouTube** с использованием cookies из браузера или файла `cookies.txt`.
- ✅ **Автоматическая установка зависимостей** при первом запуске (включая `ffmpeg`).
- ✅ **Удобный графический интерфейс** с прогресс-баром и статусом.
- ✅ **Сохранение всех файлов** в отдельные папки рядом со скриптом:
  - `YouTube_Video` — видеофайлы.
  - `YouTube_Audio` — извлечённые аудиодорожки.
  - `Transcriber` — текстовые расшифровки.

---

## 📦 Требования

- **Python 3.8+** (рекомендуется 3.10+)
- **Windows / macOS / Linux**
- **Интернет** (только для скачивания видео и моделей Whisper)

Все остальные зависимости (`yt-dlp-ytse`, `openai-whisper`, `ffmpeg`) устанавливаются **автоматически** при первом запуске скрипта.

## 🚀 Быстрый старт

1. **Скачайте скрипт** `transcriber_gui.py` и поместите его в отдельную папку.
2. **Запустите скрипт** двойным кликом или через терминал:
   ```bash
   python transcriber_gui.py
   ```
3. При первом запуске скрипт проверит и установит необходимые библиотеки. Дождитесь завершения установки.
4. Вставьте ссылку на YouTube-видео.
5. (Опционально) Укажите файл `cookies.txt`, если YouTube требует подтверждения, что вы не робот (см. раздел [Обход блокировки YouTube](#-обход-блокировки-youtube)).
6. Выберите модель Whisper (по умолчанию `small` — хороший баланс скорости и точности).
7. Нажмите **«Старт»** и дождитесь окончания процесса.

После завершения:
- Видео сохранится в папку `YouTube_Video`
- Аудиодорожка — в `YouTube_Audio`
- Расшифровка — в `Transcriber` с именем, совпадающим с названием видео.

## 🛡 Обход блокировки YouTube

YouTube активно борется с автоматическим скачиванием. Если при скачивании появляется ошибка **«Sign in to confirm you're not a bot»**, воспользуйтесь одним из способов:

### 🔐 Способ 1. Использовать cookies из браузера (рекомендуется)

Скрипт умеет автоматически извлекать cookies из **Firefox, Chrome, Edge**. Просто войдите в свой аккаунт YouTube в одном из этих браузеров — скрипт сам подхватит сессию. РЕКОМЕНДУЕТСЯ УСТАНОВИТЬ **Firefox**.

### 🍪 Способ 2. Экспортировать cookies в файл

1. Установите расширение **«Get cookies.txt LOCALLY»** для вашего браузера (доступно для Chrome, Firefox, Edge).
2. Зайдите на YouTube и войдите в аккаунт.
3. Нажмите на иконку расширения и выберите **«Export»**.
4. Сохраните файл (например, `youtube_cookies.txt`).
5. В интерфейсе скрипта нажмите **«📁»** и выберите этот файл.

### 🧩 Способ 3. Использовать форк `yt-dlp-ytse`

Скрипт автоматически устанавливает специальный форк `yt-dlp-ytse`, который содержит последние патчи для обхода защиты. Вам не нужно ничего делать вручную.

## 🎛 Выбор модели Whisper

В выпадающем списке доступны следующие модели:

| Модель  | Размер  | Скорость (CPU) | Точность | Рекомендация |
|---------|---------|----------------|----------|--------------|
| `tiny`  | ~75 MB  | Очень быстро   | Низкая   | Для тестов   |
| `base`  | ~145 MB | Быстро         | Средняя  | Для черновиков |
| `small` | ~488 MB | Умеренно       | Хорошая  | **Оптимально** |
| `medium`| ~1.5 GB | Медленно       | Высокая  | Для важных записей |
| `large` | ~2.9 GB | Очень медленно | Максимальная | Требует мощного ПК |

Модель скачивается при первом использовании и сохраняется в кэше Whisper.

## 📁 Структура папок

После запуска скрипта рядом с ним автоматически создадутся три папки:

```
📁 ваша_папка_со_скриптом/
├── transcriber_gui.py
├── transcriber_config.json      (настройки cookies)
├── 📁 YouTube_Video/            (скачанные видео .mp4)
├── 📁 YouTube_Audio/            (извлечённые аудио .wav)
└── 📁 Transcriber/              (текстовые расшифровки .txt)
```

Вы можете вручную перемещать или удалять файлы в этих папках — скрипт не будет их трогать.

## 🛠 Устранение неполадок

### ❌ Ошибка «Sign in to confirm you're not a bot»

- Убедитесь, что вы вошли в YouTube в браузере **Firefox, Chrome или Edge**.
- Либо экспортируйте файл `cookies.txt` и укажите его в программе.

### ❌ Не устанавливается `yt-dlp-ytse`

- Запустите вручную в терминале:
  ```bash
  pip install -U yt-dlp-ytse
  ```

### ❌ Не работает распознавание речи

- Проверьте, что в папке `YouTube_Audio` есть файл `.wav`.
- Попробуйте выбрать модель поменьше (`tiny` или `base`).

### ❌ Ошибка `FP16 is not supported on CPU`

- Это не ошибка, а предупреждение. Скрипт автоматически переключается на FP32. Игнорируйте его.
